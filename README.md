# ♿ Accessibility & Inclusion

SARA AI is designed for two underserved communities.

## 👁️ Visually Impaired Students

- Fully voice-controlled interface  
- No mouse or screen navigation required  
- Text-to-speech reads all content aloud  
- High contrast display mode  

## 🔤 Dyslexic Students

- OpenDyslexic reading mode  
- Increased letter and word spacing  
- Simplified AI responses  
- Voice input removes typing barriers  
- Content delivered in smaller readable sections

# 🚀 How to Run

1. Download `sara_student.html`
2. Open the file in **Google Chrome** or any modern browser
3. Allow microphone access when prompted
4. Say **"Hello SARA"** to begin

The application works **directly in the browser** and requires **no installation**.
